# Project Overview

## About This Project

This repository documents my practical learning journey in Software Quality Assurance (QA). Instead of only collecting certificates, I build a complete QA portfolio by applying testing concepts to a real web application.

The portfolio is developed progressively as I complete each learning module, following industry-standard QA practices.

---

# Application Under Test

**Application:** OrangeHRM Open Source Demo

**Application Type:** Human Resource Management System (HRMS)

OrangeHRM is an open-source Human Resource Management system that provides various business features such as employee management, leave management, recruitment, and administration. It is widely used as a practice application for Software Testing and QA Automation.

The primary goal of this project is to demonstrate my understanding of Software Quality Assurance through practical implementation, including:

# Learning Approach

Instead of creating isolated exercises, every completed learning module contributes directly to this repository.

Each new topic expands the project with a new QA artifact, allowing the repository to evolve similarly to a real QA project.
